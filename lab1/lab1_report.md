## Цель работы
Научиться работать с Docker: устанавливать Docker, создавать Dockerfile, собирать образы, запускать контейнеры и управлять ими.

## Ход работы
1. Проверка установки Docker: `docker --version`, `docker run hello-world`.
2. Скачан образ Ubuntu, запущен интерактивный контейнер, установлен `curl`.
3. Запущен контейнер с nginx (`docker run -d -p 8080:80 --name web-server nginx:alpine`), проверена работа в браузере, изучены логи и выполнен `exec`.
4. Отработаны команды управления контейнерами: `stop`, `start`, `rm`, удаление образа `nginx:alpine`.
5. Создан том `my-volume`, запущены два контейнера с монтированием тома, проверено сохранение данных между контейнерами (файл `/data/test.txt` успешно восстановлен во втором контейнере).

## Выводы
В ходе работы были освоены базовые команды Docker, получены навыки запуска контейнеров, управления ими и работы с томами. Docker успешно установлен и настроен, все задания выполнены.
